# dotfiles
 
## Installation

 ```shell
 git clone https://github.com/hens0n/dotfiles ~/.dotfiles
```

### Ubuntu

```shell
apt-get install stow
```

### Mac


```shell
apt-get install stow
```

Then simply use stow to install the dotfiles you want to use:

```shell
cd ~/.dotfiles
stow nvim
stow tmux
```
